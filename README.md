# Authentication & Key Agreement Based on Anonymous Identity for Peer-to-Peer Cloud

## 📌 Project Overview

This project focuses on providing secure communication and privacy protection in Peer-to-Peer (P2P) cloud environments. The system uses anonymous identity techniques to hide the real identity of users and ensures secure authentication between cloud users.

To improve security, the project uses the **AES (Advanced Encryption Standard)** algorithm for encryption and decryption of sensitive data. The system also includes attack detection mechanisms to identify unauthorized access and suspicious activities in the cloud environment.

The project is developed using **Java and JSP** with **Apache Tomcat Server** and **MySQL Database**.

AES (Advanced Encryption Standard) is used in this project to secure sensitive cloud data.

## AES Functions in the Project

* Data Encryption
* Data Decryption
* Secure Communication
* Data Protection from Unauthorized Access
